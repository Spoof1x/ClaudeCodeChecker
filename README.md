# ClaudeChecker

Локальная веб-панель, чтобы пачкой следить за своими аккаунтами claude.ai: проверять статус, баланс и лимиты, смотреть сессии и токены, апгрейдить тариф и пополнять баланс. Каждый аккаунт ходит через свой прокси.

Внутри всего два файла: бэкенд `app.py` (чистый Python, без зависимостей) и фронтенд `index.html`.

## Как запустить

```bash
python app.py
```

Откроется на `http://127.0.0.1:4444`. Другой порт — `--port 5000`.

Нужны только **Python 3.8+** и **curl** в системе (через него идут запросы по прокси)
